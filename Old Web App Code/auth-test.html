📋 Setup Instructions

Before testing, you need to set up OAuth credentials in Google Cloud Console:

  1. Go to Google Cloud Console: console.cloud.google.com
  2. Create a new project or select existing one
  3. Enable APIs: Google Drive API, Google+ API
  4. Create OAuth 2.0 credentials: Web application type
  5. Add authorized origins: Your domain (e.g., localhost:8000 for testing)
  6. Copy the Client ID and paste it below
